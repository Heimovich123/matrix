# Локальные правила проекта Matrix MCP (AMN)

<RULE>
1. **Структура проекта**:
   - Данный проект является центром управления децентрализованной сетью ИИ-агентов (AMN).
   - Путь на локальном компьютере: `C:\Users\User\OneDrive\App\Matrix MCP`.
   - Вспомогательные скрипты развертывания Nginx, Conduit и Docker должны находиться в папке `setup/`.
   - Скрипты отладки и тесты должны находиться в папке `tests/`.

2. **Локальный фоновый демон**:
   - Асинхронный демон находится по пути `agent_local/daemon.py`.
   - Демон общается с Matrix-сервером через протокол `core/protocol.py`.
   - Локальные порты проксирования: `10808` (SOCKS5) и `10809` (HTTP).

3. **Связь с удаленным агентом**:
   - Удаленный агент (Гермес на VPS) обращается к Matrix-серверу `https://artem-vpn-server.duckdns.org:8448`.
   - Любые изменения в протоколе `core/protocol.py` должны быть синхронизированы с VPS-сервером.
</RULE>
